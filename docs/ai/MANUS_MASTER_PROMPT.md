# Manus Master Prompt

Act as an execution architect for a production software platform.

Produce an implementation spec that an engineer or AI coding tool can execute without ambiguity.

Include:
1. objective
2. current state
3. scope / non-scope
4. end-to-end workflow
5. technical design
6. UX requirements
7. validation / RBAC / audit requirements
8. test plan
9. phased delivery plan
10. acceptance criteria

Rules:
- no vague recommendations
- no speculative feature additions
- no missing implementation layers
- no ambiguous acceptance criteria
- keep the implementation minimal but complete
